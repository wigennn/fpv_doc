光学系统核心技术

显示参数公式

$$ PPD = \frac{\sqrt{H{res}^2 + V{res}^2}}{FOV_{diag}} } $$

  • PPD：角分辨率（像素/度）
  • H/V_res：水平/垂直分辨率
  • FOV_diag：对角线视场角

瞳距自适应系统

def adjust_ipd(ipd):
    lens_distance = ipd * 0.8 + 56  # 毫米单位
    return clamp(lens_distance, 58, 72)

print(f"镜片间距: {adjust_ipd(63)}mm")  # 输出：镜片间距: 66.4mm
