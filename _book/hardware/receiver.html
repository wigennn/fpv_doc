安装与调试规范

天线部署黄金法则

  1. ​极化匹配​：接收机与遥控器天线同极化方向（垂直/水平）​
  2. 空间分集​：双天线呈90°夹角布置
  3. 避障原则​：
    • 远离碳纤维结构＞5cm
    • 与电机线距离＞10cm
    • 避免与图传天线平行

信号强度测试代码

import numpy as np

def rssi_to_percent(rssi, max_rssi=-40, min_rssi=-120):
    return np.clip((rssi - min_rssi)/(max_rssi - min_rssi)*100, 0, 100)

print(f"信号强度: {rssi_to_percent(-85)}%")  # 输出：信号强度: 54.5%
